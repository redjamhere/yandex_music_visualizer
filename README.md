# Yandex Music Anomaly Visualizer

Абстрактный визуализатор для Яндекс Музыки, который заменяет стандартную анимацию «Моей волны» на динамические эффекты, реагирующие на музыку и движения курсора.

## Особенности
* **Несколько стилей**: Anomaly (классика), Minimal Lines и Anime.
* **Настройки**: Возможность выбора стиля через страницу параметров расширения.

## Установка локально

Для запуска расширения в режиме разработчика выполните следующие шаги:

1. **Скачайте файлы проекта**: Сохраните все файлы из папки `yandex_music_visualizer` на ваш компьютер.
2. **Откройте управление расширениями**: Перейдите в Chrome по адресу `chrome://extensions/`.
3. **Включите режим разработчика**: Переключите тумблер **Developer mode** (Режим разработчика) в правом верхнем углу.
4. **Загрузите расширение**: Нажмите кнопку **Load unpacked** (Загрузить распакованное расширение).
5. **Выберите папку**: Укажите путь к папке `yandex_music_visualizer`.
6. **Готово**: Откройте [Яндекс Музыку](https://music.yandex.ru), запустите «Мою волну» и кликните в любом месте страницы, чтобы активировать визуализатор.

## Структура проекта

* `manifest.json` — метаданные расширения, разрешения и описание ресурсов.
* `content.js` — загрузчик, который определяет выбранный стиль и внедряет скрипты в страницу.
* `core.js` — основное «ядро» визуализатора: перехват аудио, управление частицами и цикл анимации.
* `styles/` — папка с модулями визуальных стилей.
* `rules.json` — правила для обхода политики CORS при анализе аудиоданных.
* `options.html` / `options.js` — интерфейс настроек расширения.

## Как добавить свой стиль (Расширение)

Проект спроектирован так, чтобы вы могли легко добавить новую визуализацию, не затрагивая основной движок.

### 1. Создайте файл стиля
Создайте новый `.js` файл в папке `styles/` (например, `styles/my_style.js`). Используйте следующий шаблон:

```javascript
window.CurrentVisualizerStyle = {
    // Основная отрисовка центрального элемента
    draw(ctx, cx, cy, intensity, time, dataArray) {
        // ctx - контекст canvas
        // cx, cy - координаты центра "Моей волны"
        // intensity - общая громкость (0.0 - 1.0)
        // time - счетчик времени для анимации
        // dataArray - массив частот (Uint8Array)
        
        ctx.beginPath();
        ctx.strokeStyle = 'white';
        ctx.arc(cx, cy, 50 + intensity * 100, 0, Math.PI * 2);
        ctx.stroke();
    }
};
